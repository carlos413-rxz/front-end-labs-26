Capítulo 1: Era só um "Hello World"...

Tudo começou inocentemente. Eu só queria fazer um "Hello World". Mas quando percebi, já estava discutindo com o CSS às 3 da manhã porque um botão NÃO queria centralizar.

😤 Sintomas comuns de um Dev Iniciante:

  • ✔ Funciona no meu computador 🤡
  • ✔ Medo de mexer no código e parar de funcionar
  • ✔ 200 abas abertas no navegador
  • ✔ Stack Overflow salvo como favorito

🔍 Fases do Debug:

  1. Negação: "O erro deve ser do computador."
  2. Raiva: "EU ODEIO PROGRAMAR!"
  3. Barganha: "Se funcionar agora, nunca mais reclamo."
  4. Aceitação: Faltava um ponto e vírgula.

Moral da história: programar é 10% escrever código e 90% procurar onde você esqueceu um detalhe minúsculo.
